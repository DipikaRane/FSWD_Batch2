DOM -- Document Object Model The whole html page we create it works like a document to the JavaScript ##DOM manipulation -- changes / manipulation can be done using three selectors >> id - unique >>className - common >>tagname -- h1, p, span, a etc if id >>> document.getElementById('id_name') e.g. : document.getElementById('QuickSearch') if className >> document.getElementsByClassName('className') e.g.: document.getElementsByClassName('option') if tagname >> document.getElementsByTagName('tagname') e.g.: document.getElementsByTagName('h1')
